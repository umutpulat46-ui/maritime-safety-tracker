# Maritime Safety Tracker 🌊

A high-performance Flutter mobile application designed for real-time vessel tracking and maritime safety monitoring.

## 🛠 Technical Highlights

Real-time Tracking: Integrated with geolocator for precision.

Background Resilience: Uses workmanager to ensure tracking continues even when the app is in the background.

Local Data Handling: Managed via sqflite for high-speed offline access.

Code Quality: Enforced through analysis_options.yaml for clean and maintainable code.

## 🚀 AI-Native Workflow
Developed with an AI-first mindset, utilizing tools like Claude 3.5, Cursor, and Gemini to accelerate development cycles and ensure rapid prototyping.

## 📈 Goal
Bridging the safety gap for small vessel owners with a "ship fast, iterate faster" philosophy.
